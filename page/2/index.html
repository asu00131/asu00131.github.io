Hexo升级思路
发表于2023-01-18
Butterfly主题升级Hexo 升级思路 检查当前 Hexo 版本：在终端中输入 hexo -v 查看当前 Hexo 版本。 更新 Hexo：在终端中输入 npm update -g hexo-cli 来更新 Hexo。 更新 Hexo 插件：在终端中输入 npm update 来更新 Hexo 插件。 更新主题：在终端中输入 cd themes/主题名，我使用的是Butterfly主题，这里就在平时发布的根目录下输入，我就输入cd D:\11040\Documents\GitHub\Hexo-butterfly-demo 然后输入 git pull 来更新主题。OK,进行1步骤，确认是最新的了。 更新 Hexo 配置文件：检查 Hexo 官方文档，查看新版本的配置文件，并将其中的新配置添加到自己的配置文件中。 重新生成静态文件：在终端中输入 hexo generate 来重新生成静态文件。
